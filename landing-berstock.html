FAQ

Pertanyaan yang Sering Ditanya

Data toko saya aman tidak?
Aman. Data disimpan di Cloudflare KV (infrastruktur kelas enterprise) dengan enkripsi end-to-end. Kami tidak share data ke pihak ketiga. Anda bisa hapus akun dan semua data kapanpun via WhatsApp ke kami.
Apa bedanya dengan ChatGPT / aplikasi lain?
ChatGPT generic — tidak tahu data toko Anda. Berstock terhubung langsung ke BerBisnis Anda, jadi jawabannya based on data REAL toko Anda, bukan halusinasi. Powered by Claude AI yang lebih akurat dari ChatGPT untuk reasoning Bahasa Indonesia.
Bagaimana cara setupnya? Saya tidak techy.
Setup 1 menit. Untuk paket Pro Early Bird, kami yang setup-kan. Anda cuma terima kode aktivasi, buka Telegram, ketik /start <kode>, selesai. Total waktu Anda: 3 menit.
Bot bisa edit data atau cuma baca?
Saat ini read-only (cuma baca). Ini intentional untuk safety — bot tidak bisa hapus barang atau cancel transaksi. Edit tetap di aplikasi BerBisnis. Phase 2 nanti kami buka feature action dengan approval workflow (tapi setelah trust terbangun).
Kalau koneksi internet putus gimana?
Aplikasi BerBisnis Anda offline-first — tetap bisa input transaksi tanpa internet. Begitu online lagi, data auto-sync ke cloud, dan bot bisa lagi diakses.
Bisa cancel kapan saja?
Bisa, tanpa penalty. Bayar bulanan, kalau dirasa tidak cocok cukup chat WA, kami matikan akun. Data Anda tetap bisa di-export dari BerBisnis (offline-first).
