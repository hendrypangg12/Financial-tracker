❓ FAQ

Pertanyaan yang sering ditanyakan

Bedanya dengan aplikasi keuangan lain?
BerUang fokus pada SIMPLE. Tidak ada feature overload. Cukup input chat atau foto struk, dashboard auto generate. Cocok untuk pemula yang baru mau atur keuangan.
Apa bedanya bulanan vs lifetime?
Bulanan Rp 35rb berlaku 30 hari. Lifetime Rp 125rb sekali bayar untuk SELAMANYA + dapat update fitur baru gratis. Lifetime jauh lebih hemat (setara ~3.5 bulan).
Apakah datanya aman?
Ya. Data tersimpan di Google Cloud Firebase yang aman, dengan enkripsi. Setiap akun terisolasi (tidak bisa diakses user lain). Hanya Anda yang bisa lihat data Anda.
Bisa dipakai di iPhone, Android, laptop?
Ya semua bisa! Cukup buka link aplikasi di browser apa saja (Safari, Chrome, dll). Login dengan email yang sama, data otomatis sync.
Cara bayarnya bagaimana?
Transfer ke BCA 7130902183 a/n HENDRY (atau request QRIS), lalu kirim bukti via WhatsApp. Akun Anda akan diaktifkan dalam 5 menit.
Garansi uang kembali?
Ya, ada garansi 3 hari pertama. Kalau merasa tidak cocok, hubungi admin via WA, uang dikembalikan 100%.
Update fitur baru ada terus?
Ya! Aplikasi terus diupdate. Semua user bulanan & lifetime dapat update gratis. Saran fitur dari user juga akan dipertimbangkan.

Siap jadi ber-uang beneran? 🚀

Mulai catat uangmu hari ini juga. Cuma Rp 35rb buat 1 bulan penuh kontrol finansial.
